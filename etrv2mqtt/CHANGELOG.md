# Changelog

All notable changes to this add-on will be documented in this file.

## [1.0.0] - 2025-10-11

### Added
- Initial release of eTRV MQTT Bridge add-on
- MQTT autodiscovery for Home Assistant
- Support for multiple Danfoss Eco thermostats
- Temperature control via MQTT
- Battery monitoring
- Configurable polling intervals
- Room temperature reporting

### Features
- Works with Mosquitto broker add-on
- Automatic device discovery
- Robust Bluetooth connection handling
- Retry logic for sleeping devices
- Comprehensive logging
