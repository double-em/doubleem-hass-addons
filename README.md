# double-em Home Assistant Add-ons

A collection of Home Assistant add-ons managed by double-em.

## Add-ons

### LLM AI Dashboard

AI Provider management, voice cloning, TTS, and AI memory for Home Assistant.

**Features:**
- AI Provider Management (MiniMax, etc.)
- Voice Cloning with OmniVoice
- Voice Design with customizable attributes
- Text-to-Speech synthesis
- Conversation with AI providers
- AI Memory with semantic search
- Home Assistant Assist integration

### eTRV MQTT Bridge

MQTT bridge for Danfoss Eco Bluetooth thermostats.

## Installation

1. Add this repository to Home Assistant:
   - Go to **Settings → System → Add-ons** → **Add-on Store**
   - Click the three-dot menu → **Repositories**
   - Add: `https://github.com/double-em/doubleem-hass-addons`
2. Find the add-on you want in the list
3. Click **Install**
4. Configure and start the add-on

## Source Code

The source code for each add-on is maintained in separate repositories:
